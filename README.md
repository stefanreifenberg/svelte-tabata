# Svelte interval timer

Project developed in [SvelteKit](https://kit.svelte.dev/). 

App can be found here [intervall.app](https://intervall.app).

Clone this repo

````
git clone https://github.com/stefanreifenberg/svelte-tabata.git
````
Install dependecies:
````
npm install
````
Start the App
````
npm run dev
````
